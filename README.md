# 区块链预测市场平台（原型）

一个可运行的预测市场平台原型，覆盖从 **市场创建 → 管理员审核（含 AI）→ 多账户挂单撮合交易 → 持仓与成交记录 → 管理员结算（含 AI）→ 收益发放** 的完整流程，并提供 Python 数据分析工作台。

## 快速开始

```bash
npm install
npm run dev
```

如果你在 Windows 上遇到 `EACCES ... :3000`，可用下面命令换端口启动（示例 3100）：

```bash
cmd /c "set PORT=3100&& npm run dev"
```
