### 关键技术点
- docker inspect 格式参数 `{{.State.Running}}` 必须用单引号变量存储，不能直接放双引号字符串
- bash 脚本中的 $ 变量（如 $ORDERER_CA）在 PS 字符串中需转义或用 [char]36 拼接
- Here-String 中含 ' 会提前结束单引号 here-string，必须用数组 -join 方式代替
- PS AST 解析器对含中文的 .ps1 文件会产生误报，不影响实际执行
- 连接配置已存在: fabric-samples/test-network/organizations/peerOrganizations/org1.example.com/connection-org1.json

### 架构决策
- **改用 Hyperledger Fabric 自建私有链**，不依赖外部 API / 测试网
- 使用 fabric-samples/test-network（已下载），Docker 部署 2org+orderer+CA
- Node.js 后端保持 fabric-network v2 SDK（已有）
